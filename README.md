#Home Assistant Add-on Collection

Welcome to my personal collection of Home Assistant Add-ons!
About this Repository

This repository serves as a central hub for various Home Assistant Add-ons that I have developed to integrate my smart home devices.

Please note that these tools were originally created for my own personal use. While I am happy to share them with the community, they are tailored to my specific setup and requirements.
Built with AI

I’m a QA, so I’m terribly bad at programming. A significant portion of the logic, structure, and code within these Add-ons was crafted with the assistance of AI. This has allowed me to turn my ideas into functional solutions faster and more efficiently.
Disclaimer

    Use at your own risk: Since these Add-ons are primarily for my private use, they might lack the extensive testing or documentation found in official integrations.

    Community-driven: These are independent, community-driven projects and are not officially affiliated with or endorsed by the manufacturers of the devices they integrate.

    Support: While I appreciate feedback and suggestions, please understand that I may not be able to provide extensive support. If you encounter issues, feel free to open a GitHub issue, and I will do my best to address it when time permits.

How to use

To use these Add-ons, add this repository to your Home Assistant Add-on Store:

    Go to Settings -> Add-ons.

    Click the three-dot menu in the top right corner and select Repositories.

    Add this repository URL: [https://github.com/Donzaffi/Home-Assistant-Addons-]

    Click Add and then Close.
